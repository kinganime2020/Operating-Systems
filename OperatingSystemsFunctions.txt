Memory Management: The processor uses a RAM chip as a way to access data and instructions that are stored temporarily on it to work on its analysis. When there is more than one program running at the same time, the operating system allocates a location in the memory intended to keep everything related to the program in the same location in order to preserve the non-leakage of data between the sites. And in case of data leakage from one location to another, the program stops working and the phrase "program" does not respond.
Processor Management: The operating system does organizing and management In a multiprogramming environment. the operating system chooses the request wherein forms approach the processor, and how much preparation time each procedure has.
